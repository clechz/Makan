MAKAN — BEFORE/AFTER SLIDER IMAGES
===================================

Drop two files in this folder:

  before.jpg  —  Raw satellite/aerial imagery (no overlays)
  after.jpg   —  Same imagery with hand-drawn intelligence overlays

REQUIREMENTS
------------
- Both images MUST be the same aspect ratio and ideally identical
  underlying scene (generate "before" first, then use img-to-img
  edit to add overlays for "after" — keeps the comparison honest)
- Aspect ratio: 16:10 or wider (matches the browser window frame)
- Minimum resolution: 2000 × 1250 px
- Format: JPG (smaller file size) or PNG if you need transparency
- No text, no numbers, no labels in either image

OVERLAY STYLE (for "after")
---------------------------
- Amber #E8A838 for alert/change markers
- Off-white #F0EDE6 for neutral AOI boundaries / waypoints
- Hand-drawn feel: thin technical-pen lines, slight line wobble
- Opacity ~80% so underlying imagery stays legible
- No UI chrome, no text — pure drawn annotation

SEE prompts in the conversation for the full generation instructions
(construction site / pipeline / urban encroachment options).
